# todo-app

## Project setup
```
npm install
npm install @heroicons/vue
npm install json-server
npm install firebase
```

### Compiles and hot-reloads for development
```
npm run serve
json-server --watch src/data/db.json
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
